## Penjelasan

Aplikasi ini merupakan hasil dari Praktikum 3 Teknologi Mobile yang membahas tentang layouting dan styling pada Flutter. Aplikasi yang dibuat bernama **Task Board**, yaitu aplikasi sederhana untuk menampilkan daftar tugas mahasiswa dalam bentuk list.

Pada aplikasi ini, layout disusun menggunakan widget `Row` dan `Column`. Widget `Column` digunakan untuk menyusun tampilan secara vertikal, sedangkan `Row` digunakan untuk menyusun komponen secara horizontal, seperti pada bagian header, filter kategori, dan item task.

Aplikasi ini juga menggunakan `Expanded` dan `Spacer` untuk mengatur ruang antar-widget agar tampilan menjadi lebih rapi dan responsif. Untuk menampilkan daftar tugas, digunakan `ListView.builder` sehingga data dapat ditampilkan secara dinamis.

Setiap item pada list memiliki icon atau avatar, judul, dan deskripsi. Selain itu, tampilan item juga diberi styling seperti padding, margin, border radius, warna, dan shadow agar terlihat lebih menarik dan mudah dibaca.

Secara keseluruhan, aplikasi ini dibuat untuk menerapkan konsep layouting dan styling pada Flutter sesuai dengan ketentuan praktikum.
